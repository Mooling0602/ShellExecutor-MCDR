# ty - Extremely Fast Python Type Checker by Astral

**ty** is an extremely fast Python type checker and language server, written in Rust by [Astral](https://astral.sh) (the makers of Ruff).

This project uses **ty** as its language server and type checker.

## Key Features

- **Blazing fast** — incremental analysis with very low latency, ideal for large codebases
- **Static type checking** — catches type-related issues before runtime
- **LSP support** — integrates as a language server into editors (VS Code, Neovim, etc.)
- **Ecosystem support** — first-class support for popular libraries like Pydantic and Django
- **Current status** — now in Beta
